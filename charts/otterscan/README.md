
# otterscan

![Version: 0.1.0](https://img.shields.io/badge/Version-0.1.0-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square)

An open-source, fast, local, laptop-friendly Ethereum block explorer. It is designed to be run locally with an archive node companion, more specifically, with Erigon.

**Homepage:** <https://github.com/wmitsuda/otterscan>

## Source Code

* <https://github.com/wmitsuda/otterscan>

# Examples

## Setting the Erigon Ethereum execution client node url required for chain data inspection

```yaml
extraEnv:
  ERIGON_URL: "http://erigon.default.svc.cluster.local:8545"
```
